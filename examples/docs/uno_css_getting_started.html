UnoCSS

UnoCSS is an on-demand atomic CSS engine built for instant design systems.

Developers compose utilities using presets, transformers, and variant groups.

Starter Workflow

  1. Install the CLI or Vite plugin and configure presets like presetUno.
  2. Create a uno.config.ts file to register shortcuts and theme tokens.
  3. Run the dev server to enjoy instant rebuilds and design token autocompletion.
