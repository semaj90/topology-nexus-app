Drizzle ORM

Drizzle ORM provides a type-safe SQL toolkit for TypeScript and JavaScript runtimes.

It emphasises schema-first workflows, lightweight runtime primitives, and fluent query builders.

Highlights

  • Declarative schema migrations stored alongside application code.
  • Support for Postgres, MySQL, SQLite, and platform adapters like Neon.
  • Intellisense-friendly query builder with full type inference.
